# Task 2 - Email Template

## Project Overview
Designed a modern promotional email template for a beauty brand using Figma. The layout focuses on visual appeal, clear product presentation, and an engaging call-to-action.

## Features
- Attractive hero banner
- Product showcase section
- Shop Now CTA button
- Social media icons
- Clean and organized layout
- Mobile-friendly design

## Tools Used
- Figma

## Deliverables
- UI Design
- Screenshot
- Demo Video
